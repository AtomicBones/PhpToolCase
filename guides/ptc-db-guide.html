Introducing The Mysql Helper Component

The mysql helper componenet is very easy to use, and can simplify the process of parsing and manipulating data with mysql queries, The class has a set of tools that can access mysql tables, and process data with just 2 or 3 lines of code. This component, gives the ability to create all types of queries, and assigns a preset reference for later use, which gives the programmer more flexibility. I especially advice this class to programmers that are new to mysql server.

Main Features:

  • Fast & esay to use, for manipulating and retrieving mysql tables
  • Ability to create select, insert, update, and delete statements
  • All methods for select statements return associatives arrays
  • Stores references of queries for later use
  • Implements mysql_real_escape_string in every query(to prevent sql injection)
  • All methods can be called statically, without initializing the class
  • Uses the PtcDebug class for logging all queries

Getting Started

To get started, download PtcDb.php from the repository and add the following lines of code to initialize the class and connect to db(make sure to change the dbConnect() parameters).

Next, will try to read all rows from a table, and will be adding order and limit to our select statement.
The return result, will be a 2 dimensions associative array with all rows defined by the "$limit" parameter.
  1. # order and limit
  2.     $order="ORDER BY `id` ASC";
  3.     $limit="LIMIT 2";
  4.     $data=$ptc->readTable("test_table","",$order,$limit);                            
  5.     print "<b>readTable() result(with order and limit)</b>";
  6.     print "<pre>".print_r($data,true)."</pre>";
That should get you started with the mysql helper component.
